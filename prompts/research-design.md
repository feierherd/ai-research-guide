title: Prompts para el diseño de investigación
---

Los siguientes prompts están diseñados para las etapas intermedias del proyecto, cuando el objetivo es refinar una pregunta de investigación, desarrollar un argumento y evaluar la viabilidad del diseño. Son más útiles cuando ya tenés una idea preliminar sobre qué querés estudiar.

---

## Refinar una pregunta de investigación

```
Mi pregunta de investigación tentativa es:
"[pegar pregunta]"

¿Podés evaluarla en función de los siguientes criterios:
- ¿es suficientemente precisa o necesita delimitación temporal, geográfica
  o conceptual?
- ¿es una pregunta descriptiva o causal?
- ¿es empíricamente contestable con los datos que podrían existir?
- ¿qué conceptos necesitan definición más explícita?
```

---

## Desarrollar un mecanismo causal

```
Mi pregunta de investigación es: "[pegar pregunta]"

Tengo una intuición sobre el mecanismo que explica el fenómeno:
"[describir mecanismo en dos o tres oraciones]"

¿Podés ayudarme a:
- articular el mecanismo con mayor precisión,
- identificar los supuestos que el argumento requiere,
- derivar al menos dos implicaciones observables que podrían usarse
  para evaluar el argumento?
```

---

## Identificar explicaciones alternativas

```
Mi argumento es: "[pegar argumento]"

¿Podés sugerir:
- explicaciones alternativas que también podrían producir el mismo
  patrón empírico que predice mi argumento,
- qué implicaciones observables distinguirían mi argumento de cada
  explicación alternativa?

Quiero anticipar las críticas antes de presentar el trabajo.
```

## Evaluar un diseño de investigación

```
Propongo el siguiente diseño para evaluar mi argumento:

[describir diseño: unidad de análisis, variable dependiente, variable
independiente, estrategia de identificación, fuente de datos]

¿Podés identificar:
- posibles fuentes de sesgo de selección,
- confusores que podrían comprometer la inferencia,
- problemas de medición en las variables principales,
- estrategias que podrían mitigar esos problemas?
```

---

## Pensar en estrategias de identificación

```
Quiero estudiar el efecto de [variable X] sobre [variable Y].

El problema es que [describir el problema de identificación: selección,
endogeneidad, confusión, etc.].

¿Podés sugerir posibles estrategias de identificación que los
investigadores usan en situaciones similares? Por ejemplo:
- experimentos naturales,
- diseños de regresión discontinua,
- diferencias en diferencias,
- variables instrumentales.

Para cada estrategia, ¿cuáles serían los supuestos que necesitaría
cumplirse en mi caso?
```

---

## Identificar fuentes de datos

```
Quiero investigar [fenómeno] en [contexto geográfico/temporal].

¿Podés sugerirme qué tipos de datos podrían usarse para estudiar
esta pregunta? Incluí:
- datos administrativos o registros públicos,
- encuestas de opinión o de hogares,
- datos generados por instituciones internacionales,
- posibilidades de recolección propia.

Para cada tipo, ¿cuáles serían los principales problemas de acceso
o de calidad que tendría que anticipar?
```

---

## Formular hipótesis comprobables

```
Mi argumento central es: "[pegar argumento]"

¿Podés ayudarme a derivar hipótesis empíricas precisas a partir de
este argumento?

Para cada hipótesis, especificá:
- la relación esperada entre las variables,
- el contexto o las condiciones en que aplica,
- qué resultado la confirmaría y qué resultado la refutaría.
```

---

## Simular la crítica de un revisor

```
Aquí está el resumen de mi propuesta de investigación:

[pegar resumen de 200–300 palabras]

Actuá como un revisor académico exigente. ¿Cuáles serían las
tres críticas más importantes a este proyecto? Para cada crítica:
- describí el problema con precisión,
- explicá por qué es relevante para la validez del diseño,
- sugerí una posible respuesta o corrección.
```
